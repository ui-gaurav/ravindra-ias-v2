Ravindra IAS v2
A specialized web platform designed for Ravindra IAS, a premier coaching institute for UPSC and State Civil Services. This version (v2) focuses on a modernized user interface, improved course management, and a streamlined student learning experience.

🚀 Key Features
NCERT-Centric Curriculum: Structured modules following the institute's signature NCERT-foundation approach.

Course Management: Support for GS Foundation, UPSC Prelims/Mains, and State-specific (HCS) courses.

VOD Integration: Video-on-Demand support for flexible, pace-based learning.

Dynamic Syllabus Tracker: Interactive breakdown of UPSC and HCS examination patterns.

Responsive UI: Optimized for both desktop and mobile learners.

🛠️ Tech Stack
Frontend: React.js / Next.js (indicated by the ui- prefix)

Styling: Tailwind CSS / Styled Components

State Management: Redux or React Context API

Version Control: Git & GitHub

📦 Installation
To get a local copy up and running, follow these steps:

Clone the repository:

Bash
git clone https://github.com/ui-gaurav/ravindra-ias-v2.git
Navigate to the project directory:

Bash
cd ravindra-ias-v2
Install dependencies:

Bash
npm install
# or
yarn install
Start the development server:

Bash
npm run dev
# or
yarn dev
